CipherSight

Experimental Facial Recognition Data Encryption

🔐 What is CipherSight?

CipherSight is a playful prototype that explores encrypting sensitive data using facial recognition. It generates a key stored in keychain, and decrypts files with that key only if it detects your face.

🧠 Features

  • Facial recognition encryption (via camera)
  • Secure encryption key storage via keyring
  • Data stored securely and only decryptable by your face
  • Client-side only (nothing gets uploaded, we promise... probably)

🧪 Technologies Used

  • Custom TKinter
  • Open-CV's LBPH face recognition
  • Fernet Encryption

🚧 Status

This app has been made, but is insecure and should not be used for real-world security. It’s part of the Supersmart Goldfish Labs where nothing is real, but everything is interesting.

Stay tuned! A prototype demo is coming soon.
